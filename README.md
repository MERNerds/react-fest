# React Fest
## Project Description

React Fest is an an application for music festivals to utlizing the following technologies:

1. React
2. MongoDB
3. JWT
4. Redux
5. Node
6. GraphQL
7. Stripe API

Challenges Faced
1. Stripe API Documentation
2. DevExtreme CSS styling override
